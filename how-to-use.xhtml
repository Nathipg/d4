HOW TO USE

Now that you had discovery this new application, you need to know what you have to do to start to use it. This tutorial is going to be made in two versions, desktop and mobile.

Mobile installation
  1. Access the app center of your smartphone (if you use Android, you should access Play Store, if you use iOS, App Store and if you use Windows Phone, Store)
  2. Type “Evernote” in the search
  3. Select the application
  4. Click on “Install”
  5. Wait until the process finish
  6. Click on the icon
Desktop installation
  1. Access the website: https://evernote.com/
  2. Click on the menu icon at the top right corner
  3. Select the option “Download Evernote”
  4. Choose your platform and download it
Create a text note
  1. Click on the plus “+” icon on bottom right
  2. Click on the "text" icon
  3. Fill the title and the note fields
  4. Choose the notebook where you want to store it
  5. Click the check icon on top left corner to finish
Create a camera note
  1. Click on the plus “+” icon on bottom right
  2. Click on the "camera" icon
  3. Align the camera with a document, blackboard, bulletin board, anything that you want to transform into a note
  4. Wait while Evernote capturing the photo, and process the better adjust
  5. Choose a type when you desire in combo box
  6. Click on “save” on top right
